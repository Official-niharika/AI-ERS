# AI-ERS Citizen App

**When Seconds Matter, Intelligence Leads.**

AI-ERS Citizen App is an AI-powered emergency reporting platform that enables citizens to quickly report incidents, share critical information, and connect with emergency response services in real time.

### Features

* Emergency alert reporting
* AI-assisted incident analysis
* Voice input support
* Location sharing
* Quick emergency tags
* Modern responsive interface

### Tech Stack

* React
* Vite
* JavaScript
* Tailwind CSS

### Purpose

The Citizen App serves as the public-facing component of the AI-ERS ecosystem, helping users report emergencies while enabling faster and smarter response coordination.
